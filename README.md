## Linux 命令操作

Windows 系统使用 DOS 内核，macOS 和 Linux 使用 UNIX，所以他们的命令行工具和命令都会有所差异，分为 CMD 和 bash，最新的 macOS 使用 zsh。
Windows 系统要下载 Git bash

## 常用 Linux 命令
